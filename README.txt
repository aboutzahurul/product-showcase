== Description ==

Simple Product Showcase plugin add product-showcase as custom post type to show product and catalog for showcase only in WordPress based site.

== Installation ==

1. Upload `product-showcase` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
